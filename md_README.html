Description

Here's the code for MultiVersio for the Desmodus Versio platform. Build using Visual Studio Code. Thanks to Emilie Gilliet (spectrings and many filters are derived from her wonderful work). Thanks to the Daisy developers for their lovely platform. And thanks to Noise Engineering for their awesome products.

More info at:

https://www.modwiggler.com/forum/viewtopic.php?t=249058

Building

  1. brew install arm-none-eabi-gcc
  1. git clone https://github.com/fluffyaudio/MultiVersio.git
  1. git clone https://github.com/electro-smith/libDaisy.git
  1. cd MultiVersio
  1. make
