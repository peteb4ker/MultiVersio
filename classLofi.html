Applies lo-fi processing to the input audio samples and generates the output samples. The lo-fi effect includes RMS calculation, envelope following, filtering, compression, saturation, and delay.

Parameters
outlThe left channel output sample.
outrThe right channel output sample.
inlThe left channel input sample.
inrThe right channel input sample.
