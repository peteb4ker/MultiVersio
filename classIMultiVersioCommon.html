Detailed Description

Interface for the common functionality of the MultiVersio system.

Interface for the MultiVersio common functionality.

This class provides an interface for the common functionality of the MultiVersio system. It includes member variables and functions that are shared among different effects.

The class contains a constructor that takes a reference to a DaisyVersio object and initializes the DaisyVersio and LedsControl member variables.

The class also includes an array of pointers to IEffect instances, which can hold up to 8 effects.

The class defines static variables for delay lines, buffers, and various audio processing modules such as reverb, filters, tone, biquad, and DC block.

It also includes member variables for mode, previous mode, sample rate, and global sample rate.

The class provides virtual functions for running the MultiVersio system and getting the current mode.

Additionally, it includes reverb parameters that are used across all effects, as well as an attack lookup table.

This class defines the common functionality that is shared among different versions of MultiVersio. It provides access to the DaisyVersio object, LED control, and various audio processing components. Subclasses of IMultiVersioCommon must implement the run() and getMode() methods.
