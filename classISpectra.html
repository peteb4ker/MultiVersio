Detailed Description

Represents the interface for handling spectra data.

This class provides member variables and functions for manipulating spectra data. It includes arrays for different scales, variables for harmonics, octaves, spread, and other parameters. The class also defines constants for the chromatic scale and other properties related to spectra.
